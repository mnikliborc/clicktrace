How to install -

Windows

If you work on Windows you can use Clicktrace installer for Windows.

Linux/Mac

Clicktrace is a Java program (Java 6 or higher required), so you can run it wherever Java Runtime Environment (JRE) is installed.

To run it on Linux or Mac just download the newest zip archive and unpack it. Clicktrace is ready to run. Just double-click "clicktrace.jar" file.

For Mac users: if on Clicktrace startup you get "Unable to register JNativeHook" message, then enable Clicktrace to take shots of your screen.
